Bertin Matrices

Bertin matrices are a means to analyze structures in a matrix using two-dimensional display techniques. Basic data structures for Bertin analysis are implemented as class based on matrix, using additional attributes.

J. Bertin: La graphique et le traitement graphique de l'information. Flammarion: Paris 1977.

A. de Falguerolles; F. Friedrich & G Sawitzki: A Tribute to J. Bertin's Graphical Data Analysis. Published in: W. Bandilla, F. Faulbaum (eds.): SoftStat '97. Advances in Statistical Software 6. Lucius & Lucius, Stuttgart 1997.

To install this package directly within R, type
   install.packages("bertin",repos="http://r-forge.r-project.org")

To install the recent package from source directly within R, type
   install.packages("bertin",repos="http://r-forge.r-project.org",type="source")
